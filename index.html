Question Answer
How do I link to an image or another HTML page that's on my computer? You need to link to the relative path. If the image or HTML page is in the folder of your current HTML file, just put the filename (i.e. "index2.html" or "image.png". Otherwise, you need to provide the relative path (i.e. "~/Desktop/index2.html").
What are classes? Why should I use them? Classes are an easy way to style multiple things at once with the same style. If you give the attribute class="mystyle" to several elements and apply a style to ".mystyle" in your CSS file, the style will be applied to all HTML elements with the class "mystyle"!
